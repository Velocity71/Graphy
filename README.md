# Graphy
A function grapher coded in Java.

This project is not yet functional.

# How to Use:
1. download the code by clicking the green "code" dropdown box and select Download ZIP.
2. Extract the downloaded ZIP file to a folder.
3. Navigate into the folder using your command prompt.
4. execute the "builder.sh" script to build the project.
5. execute the "runner.sh" script whenever you would like to run the application.

Here is what that would look like on MacOS:
1. ZIP is downloaded and automatically extracted in the Downloads folder.
2. Open the "Terminal" application.
3. Enter "cd Downloads/Graphy-main" (replacing "Graphy-main" with whatever the downloaded folder is called).
3. Enter "sh builder.sh" to build the project.
4. Enter "sh runner.sh" whenever you would like to run the application.
